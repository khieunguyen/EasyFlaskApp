# Easy Flask App
A ready to use boilerplate Flask App built on top of an [AdminLTE v3.2](https://adminlte.io/) and [Awesome icons v6.2.1](https://fontawesome.com/icons). 
FYI, AdminLTE is an open-source dashboard built on top of Bootstrap 4. 


## Let's get started
> Start the app
```bash
flask run
```

If you don't want to use the authentication module, feel free to remove the authentication module `mod_auth` at line `43` in the app `__init__.py` file.
```python
def register_blueprints(app):
    for module_name in ('home','mod_auth'
                        ):
```

### Create new user
> By default, the app will redirect guest users to the login page. If you don't have any account, feel free to click to the registration page and create new account.

|     |--- templates                      # HTML pages template used to render pages

|     |     |     |--- footer.html        # App footer
|     |     |     |--- sidebar.html       # App side bar
|     |     |     |--- navigation.html    # App navigation 
|     |     |
|     |     |--- layouts                  # App layouts (master) pages
|     |     |     |--- base.html          # base layout used by common pages
|     |     |
|     |     |--- 500.html                 # 500-page
|     |     |--- 404.html                 # 404-page
